Welcome to Your New Page

This page demonstrates the power of Django template inheritance:

  • ✅ Same header and sidebar on every page
  • ✅ Easy to change page titles
  • ✅ Consistent styling and navigation
  • ✅ No code duplication
  • ✅ Easy to maintain and update

How to create a new page:

  1. Create a new template file
  2. Extend the base template: {% extends 'base.html' %}
  3. Override the blocks you need:
    • {% block title %} - Page title
    • {% block page_title %} - Header content
    • {% block content %} - Main content
    • {% block extra_css %} - Additional CSS
    • {% block extra_js %} - Additional JavaScript
